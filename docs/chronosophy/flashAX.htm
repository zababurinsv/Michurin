Реклама в Интернет

Инсталляция Macromedia Flash Player 4

В том случае, если анимированный объект будет невидим в браузере (просматривается только черный квадрат), необходимо проинсталлировать автоматически предлагаемый Macromedia Flash Player. Учитывая, что размер программного средства не превышает 217 килобайт, то время ожидание его загрузки с сервера компании производителя на скорости 14,4 килобит в секунду составит около 9 минут.

Если вы предпочитаете самостоятельно инсталлировать программное средство загрузите самораскрывающийся инсталлятор Macromedia Flash Player версии 5 (337 KB) как объект Active-X для Microsoft Internet Explorer 3.0x, Microsoft Internet Explorer 4.0x and Microsoft Internet Explorer 5.0x for Windows 95, Windows 98, Windows 2000 or Windows NT.

Системные требования для применения объект Active-X

  • IBM-совместимый персональный компьютер PC с процессором 486/25 или выше. Pentium предпочтителен.
  • Операционная среда Windows 3.1x, Windows 95, Windows 98, Windows 2000 или Windows NT.
  • Видеокарта VGA with 8-bit color (256 цветов) или больше.
  • 8 MB RAM/6 MB свободного дискового пространства.
  • браузер Microsoft Internet Explorer 3.0 или выше.
  • Требуется аудиокарта для 486 с FPU или Pentium.
