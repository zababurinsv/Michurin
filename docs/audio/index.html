Телепередачи цикла "Этносы Земли" были созданы в 1993 году на Санкт-Петербургском телевидении (СПб-5) на основе видеозаписей выступлений Л.Н. Гумилева на Ленинградском телевидении в 1988 году.

Руководство общероссийского телеканала ⌠Культура■ (ФГУП ГТРК "Культура") с 2005 г. ведет работы с ОАО "ТРК Петербург" и мэрией СПб по показу и приобретению всего цикла телепередач "Этносы Земли". После того, как пройдет показ телепередач, соблюдены юридические права (авторские и смежные), только тогда, при условии наличия соответствующего финансирования, возможно будет начинать работу по тиражированию самих передач на современных носителях.

Ниже размещены материалы любезно предоставленные Общественной организацией "Фонд Л. Н. Гумилева".

"Телевизионные передачи с записями лекций Л.Н.Гумилева". Анонс

Этнологическая карта современного состояния мира. Текст из телевизионных передач из цикла "Этносы Земли" (СПб - 5 канал, 1993 г. - запись 1988 г.) - "Природа суперэтноса" (No 25), и "Блуждающий суперэтнос" (No 20) эфиры 1994, 1995 гг.

Голос Льва Николаевича (звукозапись телепередачи на Ленинградском телевидении) в формате Real Audio (328 KB).

Формат Real Audio является одним из наиболее распространенных форматов для звукозаписи на Интернете. На Вашем компьютере еще не установлен Real Audio Player, но имеется Windows 98 или Windows 95 OSR 2, то его можно проинсталлировать, войдя в Control Panel => Add/Remove Programs => Windows Setup => Internet tools => Real Audio Player. Иначе можно загрузить эту программу с сервера фирмы его выпускающей - http://www.real.com.
